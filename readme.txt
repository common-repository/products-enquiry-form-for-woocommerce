A Simple and Lightweight plugin to add enquiry form with WooCommerce Products.

<br>
<h2><a href="http://www.wlm.webcodingplace.com/product/test-product/">LIVE DEMO</a></h2> 

<h1>AMAZING FEATURES</h1>
<ul>
<li>Sending Emails with Subject</li>
<li>Show Error and Success message after submit the form</li>
<li>Product Name and Product Url also send with email</li>
<li>Automatically add Tab in product area after activated plugin</li>
<li>Responsive Form</li>
<li>Multiple Color Handling</li>
<li>Change Form Title</li>
</ul>
